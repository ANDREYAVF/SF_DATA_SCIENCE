## Запуск приложения
В терминале выполните: streamlit run app.py
Откроется веб‑интерфейс по адресу http://localhost:8501.

## Что реализовано
Календарь и фильтры

Выбор городов (multiselect).

Диапазон дат (date_input).

Таблица данных

Отображение с подсветкой максимумов.

Автоматическая фильтрация по выбору пользователя.

Производные столбцы

temp_category: холодно/умеренно/тепло/жарко.

precip_level: без осадков/небольшие/сильные.

comfort: комфортно/некомфортно (на основе температуры, ветра и осадков).

EDA

Гистограммы, boxplots, KDE для каждого признака.

Сравнение городов через столбчатую диаграмму.

Временные ряды

График температуры со скользящим средним (7 дней).
