## Загрузка CSV‑файла

Через st.file_uploader с ограничением на .csv.

Кеширование загрузки (@st.cache_data) для ускорения.

## Отображение таблицы

st.dataframe() с фиксированной высотой.

Вывод базовой статистики о данных.

## Выбор столбцов

st.selectbox() для выбора столбцов для анализа и визуализации.

## Статистический анализ

Среднее, медиана, СКО — только для числовых столбцов.

Проверка типа через pd.api.types.is_numeric_dtype().

## Построение графиков

Линейный график, диаграмма рассеяния, столбчатая диаграмма.

Обработка ошибок при построении.

## Обработка типов данных

Числовые: гистограмма + KDE, boxplot.

Категориальные: столбчатая диаграмма частот.

## Кнопка загрузки графика

st.download_button() для сохранения текущего графика в PNG.

## Состояние сеанса и кеширование

@st.cache_data для загрузки данных.

Ключи (key) для виджетов, чтобы сохранять состояние при ререндере.
